Mute Don't play the track.
Solo Like record but additionaly prevents recording on other tracks.
Trackname The name of the track. A click on the name will select this track in the Track Editor.
Record Record on this track. Data from the tracks input device is forwarded to the tracks output device.
This means that each track is assigned an
  • Input Device
    I.e. a video camera or a MIDI keyboard.
  • Output Device
    I.e. a Monitor window or a MIDI outchannel.
Monitor Don't record but forward data from the tracks input device to the tracks output device.
R/W Don't know yet. It's a feature of Steinbergs CuBase(TM)).
Lock Don't modify this track.
