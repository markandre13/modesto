Modesto - Introduction

[ prev ] [ up ] [ next ]

Modesto is still under construction and currently only able to

  • Load and play MIDI files on /dev/midi (default) and /dev/ttyS0
  • Mute tracks
  • Add tracks
  • Edit some track parameters (name, program, channel)
  • Selecting input and output channels
  • Monitor a MIDI track
  • Record, store and play MIDI and Audio Tracks
  • MIDI isn't stored in MIDI files.
  • Audio is stored to and read from 44100Hz, 16 Bit, stero WAV Files
  • automatically increase channel when creating a new midi track
  • midi tracks for the same channel should share the program

Some known issues to be fixed next:

  • mmap needed when playing audio tracks?
  • monitor doesn't work on audio

Tips

Audio

Modesto uses ALSA for audio input and output. It doesn't provide access to the mixer controlls but another program like alsamixer can be used instead.
The space bar will toggle capturing (aka recording) and the 'M' key will toggle muting.

With the following option, microphone input will not be send directly to the soundcards output but is routed through Modesto:

  • Mic: Capture & mute on
  • AD Input: Set mute to on

You may also want set "Mic Boost" to on (by disabling) mute.
